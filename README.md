# Flights Board Management (React + TypeScript)

A simple flights board management UI built with React, TypeScript, and Tailwind CSS.  
Features:
- Flights table with server-side filtering by destination and status
- Add/delete flights
- Real-time updates via SignalR
- Form validation for creating flights
